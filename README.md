# ⚽ FIFA 2026 Smart Stadium Command Center

An AI-powered Smart Stadium Operations Platform designed for the FIFA World Cup 2026. The platform enables tournament organizers, stadium operators, and security teams to monitor stadium operations, crowd movement, sustainability metrics, and fan experiences through an interactive real-time dashboard powered by Artificial Intelligence.

## 🚀 Features

- 🔒 AI-Powered Security Monitoring
  - Real-time incident dashboard
  - Crowd density monitoring
  - AI-generated incident summaries
  - Emergency alert simulation

- 🚉 Logistics & Transit Dashboard
  - Transit infrastructure visualization
  - Stadium ingress/egress analytics
  - Traffic simulation
  - Host city comparison

- 🌱 Sustainability & Digital Twin
  - Carbon footprint monitoring
  - Energy consumption tracking
  - AI energy optimization
  - Smart stadium digital twin

- 🎟 Fan Experience & Accessibility
  - Environmental Stress Index (ESI)
  - AI multilingual announcements
  - Accessibility support
  - Stadium service recommendations

- 🤖 GenAI Command Center
  - Natural language querying
  - AI-powered operational insights
  - Incident analysis
  - Intelligent recommendations

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/VISHESH-53/Challenge-4.git
```

Navigate into the project

```bash
cd Challenge-4
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Run production

```bash
npm start
```
